# 2d横版平台跳跃游戏

## 简介

这是一个使用 Godot 4.4.1 引擎独立开发的2d横版平台跳跃游戏的demo。这是我通过各种Godot引擎教程进行自学而制作的demo，旨在通过项目学习并巩固编程能力、熟悉游戏开发流程。

## 当前核心功能

- 角色状态机：实现玩家角色在不同的条件下在各种状态间的切换，并处理输入和物理碰撞
- 受击反馈：通过"HitFlash"节点和 Shader 实现角色受伤的闪光效果。
- 重生点管理：通过设计"Spawn"节点灵活管理玩家的复活位置
- UI界面：包含一个简易的开始界面和退出功能
- 地图切换：通过"MapSwitch"触发器，支持不同场景的切换

## 正在研究的功能

- 大场景内的区块加载系统
- 收集系统
- 音乐及音效
